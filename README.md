# BrogiMatAssistance

Osobní asistent pro Pavla — přebírá operativu (emaily, faktury, připomínky, odpovědi).
Pavel potvrzuje rozhodnutí. Asistent vykonává.

## Stack
- Docker na NUC (ASUS i5, 64GB RAM, 4TB)
- assistance-postgres + assistance-chroma
- Mac Agent na Apple Studiu (AppleScript konektory)
- PWA: assist.brogimart.cz
- Telegram Bot 1 (notifikace) + Bot 2 (příkazy)
- Claude Haiku (třídění) + Claude Sonnet (odpovědi, Pattern Mirror)
- iOS Shortcuts → location webhook
